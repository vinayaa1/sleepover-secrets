# sleepover secrets

# link to play: https://vinayaa1.github.io/sleeepover-secrets/

a browser-based interactive story game about a sleepover that slowly turns unsettling.

## about the game

you arrive at a sleepover expecting a normal night with a friend. as the story progresses, small details begin to feel wrong. your choices shape how suspicious the situation becomes and determine which ending you reach.

## features

- interactive branching story
- suspicion-based progression system
- multiple endings
- scene-based visual mood changes
- lightweight browser game (no installs required)

## how to play

open the game in your browser and use the on-screen buttons to make choices. each decision changes the story and can lead to different outcomes.

## endings

the game includes multiple endings depending on your choices:
- normal ending
- brave ending
